1.-entra a la carpeta auto-siiau
2.-abre el archivo auto-siiau.exe
3.-ingresa con tu codigo y nip de alumno UDG

■.-Datos Generales: Aqui podras ver algunos de tus datos mas relevantes

■.-Kardex: Todas mas materias cursadas, ya sean aprobadas o reprobadas

■.-Boleta: Materias en curso

■.-Cursos Posibles: lista de materias aun no cursadas

■.-Oferta: Acceso a la oferta academica

■.-Registro: Acceso automatico al registro de siiau
Formato de uso:
	1.-Seleccionar tu centro universitario: por defecto es CUCEI
	2.-Seleccionar el calendario actual para poder agendar la materi "en esta version no es valido seleccionar el calendario "20 B"
	3.-Rellenar los NRC con materias de preferencia
	4.-Si modo turbo esta actividad se ejecutara de forma constante en segundo plano, en caso contrario se ejecutara una sola vez y mostrara las materias seleccionadas en la agenda.
	5.-Seleccionar el tipo de registro al cual se desea acceder "En este momento solo Movilidad muestra las materias"


Importante: para hacer las pruebas nocesito que ejecuten en modo turbo y lo dejen el mayor tiempo posible, cuando den click en (STOP) se detendra y abra un archivo de texto plano con extension (.siiau)
